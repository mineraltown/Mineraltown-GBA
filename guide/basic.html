基本操作

掌机操作

键位用途

  • 方向键 走
  • A 谈话/调查/拾起面前持有物/丢下手中持有物
  • B 使用道具/持有物放入背包
  • L 口哨
  • R 跑步
  • SELECT 资产列表
  • START 主菜单

快捷键

  • A+B 打开背包
  • L+A/A+L 正着/反着从背包中拿出物品
  • L+B 交换装备/吃东西（按住L不放可以按方向键继续切换装备）
  • L+START 打开镇子地图
  • L+SELECT 打开牧场地图

电脑模拟器键位

VisualBoyAdvance模拟器默认情况下的操作

键位用途

  • 方向键 => 十字键
  • Z => A
  • X => B
  • A => L
  • S => R
  • Backspace => SELECT
  • Enter => START
